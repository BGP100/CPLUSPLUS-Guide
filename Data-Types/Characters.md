The char data type is used to store a single character. The character must be surrounded by single quotes, like 'A' or 'c':
<pre>
char myGrade = 'B';
cout &lt;&lt; myGrade;
</pre>
Alternatively, you can use ASCII values to display certain characters:
<pre>
char a = 65, b = 66, c = 67;
cout &lt;&lt; a;
cout &lt;&lt; b;
cout &lt;&lt; c;
</pre>
